Seletores CSS

Bem-vindo

No CSS nós possuimos os seletores por ID

  • Seletor por Tipo: elentos por uma tag HTML específica
  • Seletor por ID (#): Busca elementos através do atributo 'id'
  • Seletor Universal(*):Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]):Seleciona elemntos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor específico
