CSS Grid Layout

A grid layout consists of a parent element, with one or more child elements.

The parent element is called the grid container, and the child elements are called grid items.

To create a grid layout, you must first define a container element as a grid with the display property:

        .container {
            display: grid;
        }

This will turn the container into a grid container, and its direct children will become grid items.

By default, a grid layout has one column and as many rows as necessary to fit the content.

You can define the number of columns in the grid by using the grid-template-columns property:

        .container {
            display: grid;
            grid-template-columns: auto auto auto;
        }

This will create a grid with three columns of equal width. You can also use other units like pixels (px), percentages (%), or fractions (fr) to define column widths.

For example, to create a grid with two columns where the first column takes up 1/3 of the available space and the second column takes up 2/3, you can use:

        .container {
            display: grid;
            grid-template-columns: 1fr 2fr;
        }

You can also define rows in a similar way using the grid-template-rows property.
